Wake actions

When a wake phrase is detected in a transcript, the configured command runs with the post-wake text in $transcript. Useful for routing captured intent to an agent (hermes-agent --input "$transcript"), creating reminders, or anything else a shell command can do.

Template variables: $transcript (post-wake text), $transcript_full (entire transcript), $conversation_id, $wake_phrase. All shell-escaped via shlex.quote, so quotes in the transcript can't break the command.
